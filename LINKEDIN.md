Outside my day job, I build Lattice Momentum — an LLM research platform for systematic equity analysis (educational, no live trading). The LLM layer is a multi-provider Go abstraction (Anthropic, OpenAI, Gemini) over raw net/http — no SDKs — so I control Anthropic prompt caching and SSE streaming semantics directly, with per-call token costs surfaced as Prometheus metrics.

Each prompt iteration is content-hash versioned (v7-<8-char SHA-256>) and evaluated by per-ticker recommended-vs-rejected attribution: for every ticker the LLM saw, the system tracks forward returns and realized R/R, distinguishing tickers the LLM recommended from those it rejected. Outcomes feed a pgvector RAG layer that injects historical analogues into subsequent evaluations (verified-only, to keep unverifiable setups out of the retrieval pool). Tickers are anonymized pre-call to suppress name bias, and output is parsed through a JSON-contract parser with regex fallback.
